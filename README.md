\# Retail Pricing Analytics



\## Overview

This project analyses how product pricing affects demand and revenue using a simple price elasticity model.



\## Objective

Estimate price elasticity and simulate how different prices impact expected revenue.



\## Files

\- `generate\_realistic\_retail\_data.py` — generates a synthetic retail pricing dataset

\- `pricing\_analysis.py` — runs the pricing analysis and estimates elasticity

\- `price\_vs\_demand.png` — chart showing the relationship between price and demand



\## Methods

\- Synthetic data generation

\- Log-log price elasticity modelling

\- Revenue simulation

\- Data visualisation with matplotlib



\## Key Result

The model estimated a negative price elasticity, showing that demand falls as price increases.



\## Tools

\- Python

\- pandas

\- numpy

\- matplotlib



\## Business Value

This type of analysis helps retail teams:

\- understand customer price sensitivity

\- test pricing strategies

\- improve revenue decisions
